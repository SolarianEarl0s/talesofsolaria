# Images Directory

This directory contains all the image assets for the Linktree clone template.

## Required Images

Please add the following images to this directory:

- **logo.png** - Your profile picture or logo (recommended: 100x100 pixels)
- **favicon.png** - Website favicon (recommended: 32x32 pixels)
- **linkedin.png** - LinkedIn icon (recommended: 20x20 pixels)
- **github.png** - GitHub icon (recommended: 20x20 pixels)
- **instagram.png** - Instagram icon (recommended: 20x20 pixels)
- **facebook.png** - Facebook icon (recommended: 20x20 pixels)
- **credly.png** - Credly icon (recommended: 20x20 pixels)
- **hackerrank.png** - HackerRank icon (recommended: 20x20 pixels)
- **replit.png** - Replit icon (recommended: 20x20 pixels)

## Image Recommendations

- Use PNG format for transparency
- Keep file sizes small for better performance
- Icons should be at least 20x20 pixels
- Social media icons work best with solid colors
